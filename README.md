## 📌 Project Overview

Employee Attrition Intelligence Platform (EAIP) is a production-ready HR Analytics application built using Machine Learning and Streamlit.

The application predicts whether an employee is likely to leave the organization, explains the prediction using Explainable AI (SHAP), and generates actionable HR recommendations to improve employee retention.

Designed with a Microsoft Power BI-inspired interface, the platform combines predictive analytics, interactive dashboards, and business intelligence into one application.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/AkshayShukla30/Employee-Attrition-Intelligence.git

cd Employee-Attrition-Intelligence
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app/streamlit_app.py
```
